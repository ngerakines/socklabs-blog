    FluxHP is open source
---

Tonight I cleaned up and released the FluxHP project on Google Code. FluxHP
is an web application written in Erlang to manage raid DKP/HP. It uses
ErlyWeb over Yaws + Mnesia. There is plenty of documentation and a
quick-start guide to getting it running.

[http://code.google.com/p/fluxhp/](http://code.google.com/p/fluxhp/)
